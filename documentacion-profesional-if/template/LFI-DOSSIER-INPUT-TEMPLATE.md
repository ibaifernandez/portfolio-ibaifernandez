# LFi Dossier Input Template

## Objetivo

Esta plantilla existe para convertir tu memoria profesional sobre **LFi** en material utilizable para:

1. reescribir el dossier de `lfi.html`,
2. diseñar una versión visual premium sin humo,
3. extraer claims públicos creíbles,
4. reunir pruebas y assets,
5. separar lo que se puede afirmar de lo que conviene matizar o callar.

La idea no es que la rellenes "bonito". La idea es que la rellenes **rápido, en bruto y con verdad**.

---

## Cómo usar esta plantilla

- Mantén todo lo que ya aparezca como `[extraído]` si es correcto.
- Corrige lo que esté mal.
- Completa lo que esté como `[por completar]`.
- Marca lo sensible como `[confidencial]`.
- Si un dato no es exacto pero sí aproximado, escríbelo como rango o estimación.

Formato recomendado para ampliar:

```md
- [confirmado] dato sólido y publicable
- [aprox] estimación razonable
- [confidencial] no publicable, solo para orientar narrativa
- [duda] recordar / contrastar
```

### 5.1 Contexto

- [extraído] Cliente inmobiliario chileno.
- [extraído] Llevabas su campaña mensual.
- [extraído] Existían múltiples proyectos y múltiples bases de datos adscritas a cada uno.

### 5.2 Qué mejoraste

- [extraído] Reducción enorme del tiempo dedicado a ejecutar la campaña mensual.
- [extraído] Ordenaste flujos de información, recepción y procesamiento de assets y comunicación con cliente y otros departamentos.
- [extraído] En vez de subir todas las bases de datos cada mes, creaste un sistema para cargar solo leads nuevos y dar de baja los que ya habían comprado o salido del proceso.
- [extraído] Esto aportó consistencia y mejor manejo de leads.
- [extraído] También propusiste sustituir Linktree por un centralizador propio: `http://pocuro.cl/enlaces`.

### 5.3 Lead Management Automation for POCURO

#### Contexto y reto

- [extraído] 60.000+ leads mensuales desde paid media, sobre todo **Meta** y **Google Ads**.
- [extraído] Menos del **5%** tenía intención real de compra.
- [extraído] El equipo comercial no tenía capacidad para procesar ese volumen eficientemente.

#### Estrategia

- [extraído] Detectaste que **Google Ads** producía leads de mayor calidad que Meta.
- [extraído] Recomendaste reubicar presupuesto hacia Google.
- [extraído] Diseñaste e implementaste un **chatbot de WhatsApp** construido con **n8n** y una base de conocimiento propia.
- [extraído] Desarrollaste lógica de **lead scoring** para clasificar y priorizar leads por intención y fit.

#### Ejecución

- [extraído] El chatbot contactaba todo el volumen mensual.
- [extraído] El flujo filtraba, recogía intención, puntuaba y generaba archivos estructurados para asesores.

#### Resultados

- [extraído] Reducción aproximada del flujo mensual: **60k -> 6k**.
- [extraído] Filtrado de aproximadamente **500 compradores de alta intención por mes**.
- [extraído] Reducción de leads no cualificados en torno al **90%**.
- [extraído] Mejora de eficiencia comercial.
- [extraído] Mejora de ROI de paid media.
- [extraído] Disminución del coste de adquisición.
- [extraído] Solución escalable sin ampliar headcount.
